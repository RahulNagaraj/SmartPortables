How to run SmartPortables

1. Download and install Tomcat v9.0
2. Download and install MySQL v8.0
3. Download and install MongoDB v3.4
4. Download and unzip the smartportables
5. Copy the smartportables folder into Tomcat9.0\webapps
6. Ensure to change the MySQL username and password in WEB-INF\classesMySqlDataStoreUtilities.java
7. Visit localhost:8080/smartportables in your browser
